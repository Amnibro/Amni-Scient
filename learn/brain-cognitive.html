WHAT’S IN THIS CATEGORY

Cognitive-test games for adults built on the standard psychometric paradigms used in working-memory research, attention assessment, and executive-function testing: card-pair concentration, digit-span (Number Memory), N-back, Chimp Test (visual working memory), simple reaction time, mental arithmetic blitz, and the Stroop interference task.

HOW IT HELPS LEARNING

Each game uses scoring conventions that let users track personal progress: digit span by maximum length recalled, N-back by hit rate at each level, reaction time by ms median, Stroop by congruent-vs-incongruent latency. The Chimp Test reproduces the Inoue & Matsuzawa (2007) protocol where digits flash briefly then disappear; users tap them in numerical order. Speed Math escalates difficulty until the user can no longer keep pace, providing a self-calibrating challenge.

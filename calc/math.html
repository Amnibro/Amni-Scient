ENGINEERING MATH REFERENCE

CALCULUS IDENTITIES, TAYLOR SERIES, INTEGRALS, VECTORS

WHAT IT COMPUTES

Quick reference for the math that underlies engineering calculations: standard derivatives and integrals, Taylor series expansions for common functions, vector operations (dot, cross, gradient, divergence, curl), matrix operations including inversion and eigenvalues, complex number arithmetic in rectangular and polar form, and Laplace transforms for control-system analysis. Includes common trig identities and the small-angle approximations used in vibration and beam-bending derivations.

KEY EQUATIONS

  • d/dx (sin x) = cos x; d/dx (e^x) = e^x
  • ∫ 1/x dx = ln|x| + C
  • ∫ sin² x dx = x/2 - sin(2x)/4 + C
  • sin x ≈ x - x³/6 (small angle)
  • A · B = |A||B|cosθ; |A × B| = |A||B|sinθ
  • e^(jωt) = cos(ωt) + j·sin(ωt)

BEAM EQUATIONS

Bending Stressσ = M·y / I
Shear Stress (Beam)τ = V·Q / (I·b)
Deflection (SS, center pt)δ = P·L³ / (48·E·I)
Deflection (Cantilever, end)δ = P·L³ / (3·E·I)
Deflection (SS, UDL)δ = 5·w·L⁴ / (384·E·I)
Deflection (Cantilever, UDL)δ = w·L⁴ / (8·E·I)
Moment of Inertia (Rect)I = b·h³ / 12
Moment of Inertia (Circle)I = π·d⁴ / 64
Section ModulusS = I / c
Radius of Gyrationr = √(I / A)
Fixed-Fixed (center pt)M_max = P·L / 8, δ = P·L³ / (192·E·I)
Propped Cantilever (UDL)R_prop = 3wL/8, M_fix = wL²/8

STRESS / FAILURE

Von Mises (2D)σ_vm = √(σx²−σxσy+σy²+3τxy²)
Von Mises (3D)σ_vm = √½[(σ₁−σ₂)²+(σ₂−σ₃)²+(σ₃−σ₁)²]
Trescaτ_max = (σ₁−σ₃) / 2
Principal Stressσ₁,₂ = (σx+σy)/2 ± √[((σx−σy)/2)²+τxy²]
Mohr's CircleC = (σx+σy)/2, R = √[((σx−σy)/2)²+τxy²]
Factor of SafetyFoS = S_y / σ_applied
Rankine (Brittle)σ₁ ≤ S_ut, |σ₃| ≤ S_uc
Coulomb-Mohrσ₁/S_ut − σ₃/S_uc ≤ 1

FATIGUE / ENDURANCE

Endurance LimitSe' ≈ 0.5·Su (steel, Su ≤ 1400 MPa)
Marin EquationSe = ka·kb·kc·kd·ke·kf · Se'
Goodmanσa/Se + σm/Su = 1/n
Soderbergσa/Se + σm/Sy = 1/n
Gerberσa/Se + (σm/Su)² = 1/n
Basquin (S-N)σ_ar = σ_f'·(2N)^b
Fatigue Notch FactorKf = 1 + q(Kt − 1)
Miner's RuleΣ(ni/Ni) = 1 (damage sum)

COLUMN BUCKLING

Euler Critical LoadP_cr = π²EI / (KL)²
Effective LengthL_eff = K·L (K=0.5,0.7,1,2)
Slenderness Ratioλ = KL/r
Transition Slendernessλ_c = √(2π²E / Sy)
Johnson ParabolaP_cr = A·Sy[1 − (Sy(KL)²)/(4π²EI)]
AISC (ASD)F_a per AISC 360 Ch. E

BOLT / FASTENER

Bolt Load (w/ preload)F_b = F_i + C·F_ext
Joint LoadF_j = F_i − (1−C)·F_ext
Torque (K-factor)T = K·F_i·d
Tensile Stressσ = F / A_t
Interaction RatioIR = (σ/S_p)² + (τ/(0.577·S_p))²
Proof LoadF_proof = S_p · A_t
Thread ShearA_s = π·d·n·L_e·0.5
Embedding LossF_z = f_z · (n_i interfaces)

SPRING

Spring Ratek = G·d⁴ / (8·D³·N_a)
Shear Stressτ = 8·F·D / (π·d³)
Wahl FactorK_w = (4C−1)/(4C−4) + 0.615/C
Deflectionδ = F / k
Solid LengthL_s = N_t · d
Natural Frequencyf_n = (d/(2πD²Na))·√(Gg/32ρ)
Belleville LoadF = (4E·t)/((1−ν²)·K₁·D²) · δ·[(h−δ)(h−δ/2) + t²]
Spring EnergyU = ½kδ²

PRESSURE VESSEL

Hoop Stress (Thin)σ_h = p·r / t
Axial Stress (Thin)σ_a = p·r / (2t)
Hoop Stress (Thick)σ_h = (p_i·r_i² − p_o·r_o²)/(r_o²−r_i²) + r_i²r_o²(p_i−p_o)/((r_o²−r_i²)r²)
Sphere Stressσ = p·r / (2t)
ASME Min Thicknesst = pR / (SE − 0.6p)
Barlow's FormulaP_burst = 2·S·t / D

THERMAL / HEAT TRANSFER

Thermal Strainε_T = α·ΔT
Thermal Stressσ_T = E·α·ΔT
Conduction (Fourier)q = −k·A·(dT/dx)
Convection (Newton)q = h·A·(T_s − T_∞)
Radiationq = ε·σ·A·(T₁⁴ − T₂⁴)
Thermal ResistanceR = L/(k·A) (conduction)

SEAL / O-RING

Squeeze %S = (d − G_d − gap) / d × 100
Mooney-Rivlin Stressσ = 2(C₁₀+C₀₁/λ)(λ−1/λ²)
Hardness → ModulusE = 0.0981(56+7.62S)/(0.138(254−2.54S))
Contact Widthw = 2√(d·δ)
Shape Factor (Face)SF = A_load / A_free = w_c / (π·d_s − 2·w_c)
Volume FillV_fill = (π·r²) / (G_d·G_w) × 100

GEAR / POWER TRANSMISSION

Lewis Bendingσ = Wt / (F·m·Y)
AGMA Contactσ_c = C_p·√(Wt·Ko·Kv·Ks·Km / (d·F·I))
Power-TorqueP = T·ω = T·2πn/60
Gear Ratioi = N₂/N₁ = ω₁/ω₂
Belt DriveF_eff = F₁ − F₂ = T/r
Chain RatingHP = T·n / 63025 (imperial)

WELD DESIGN

Fillet Throata = 0.707·w
Weld Shearτ = F / (a·L_w)
Allowable (E70xx)F_w = 0.6 × 70 ksi = 21 ksi
Weld Polar JJ_w = Σ(Ix+Iy) per unit weld
Combined Stressτ_r = √(τx² + τy²)
Min Fillet SizePer AISC Table J2.4 (by thickness)

MATERIALS / PROPERTIES

Hooke's Lawσ = E·ε
Shear ModulusG = E / (2(1+ν))
Bulk ModulusK = E / (3(1−2ν))
Poisson's Ratioν = −ε_lateral / ε_axial
True Stressσ_T = σ_eng(1 + ε_eng)
Ramberg-Osgoodε = σ/E + (σ/K')^(1/n')
Resilienceu_r = Sy² / (2E)
Toughnessu_T ≈ (Sy+Su)/2 · ε_f

BernoulliP + ½ρv² + ρgh = const

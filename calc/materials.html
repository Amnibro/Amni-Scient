WHAT IT COMPUTES

The materials database provides design properties for common engineering materials: structural steels (A36, 1018, 4140, 4340), stainless (304, 316, 17-4 PH), aluminums (6061-T6, 7075-T6, 2024-T3), copper alloys, titanium grades, and engineering polymers (HDPE, PEEK, PTFE, nylon, polycarbonate). Each entry includes yield strength Sy, ultimate Sut, elastic modulus E, density ρ, thermal conductivity k, specific heat cp, coefficient of thermal expansion α, and Poisson's ratio ν.

KEY EQUATIONS

  • σ_allow = Sy / FoS (typical 1.5-3.0)
  • ΔL = α · L · ΔT (thermal expansion)
  • c = sqrt(E/ρ) (longitudinal sound speed)
  • κ = k / (ρ·cp) (thermal diffusivity)

References & StandardsMIL-HDBK-5J, ASM Handbook Vol. 1-2, ASTM material specs

WHEN TO USE THIS MODULE

Reference for any sizing or analysis calculation. The values are typical room-temperature properties from MIL-HDBK-5 and ASM Handbook; for high-temperature service or critical applications, consult the actual specification (ASTM, AMS, ASME) for guaranteed minimums and temperature derating curves. The "allowable stress" is typically Sy/FoS where FoS is set by application code (1.5-3 for static, 4-6 for fatigue).
